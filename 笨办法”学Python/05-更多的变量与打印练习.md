
# 🐍 Python 基础：格式化字符串 (f-string)

### 1. 什么是格式化字符串？

> [!ABSTRACT] Python 基础：格式化字符串 (f-string)
字符串是程序向人们展示信息的主要方式。**f-string**（格式化字符串字面量）是 Python 3.6 引入的一种极其简洁的语法，允许我们将变量直接嵌入到文本中。

---

### 2. 基本语法与规则
- **前缀 `f`**：在字符串的引号前添加字母 `f` 或 `F`。
- **占位符 `{}`**：在字符串内部使用大括号 `{}` 包裹变量名，Python 会自动将其替换为变量的值。

---

### 3. 实战案例：个人特征描述

通过定义变量并使用 f-string，可以快速生成易读的个人描述。

|**变量名**|**示例值**|**打印示例**|
|---|---|---|
|`my_name`|`"xiaofan"`|`f"让我们来读读{my_name}"`|
|`my_age`|`30`|`f"他的年龄是{my_age}"`|
|`my_eyes`|`"蓝色"`|`f"他的眼睛是{my_eyes}"`|
|`my_hair`|`"棕色"`|`f"他的头发是{my_hair}"`|

---

### 4. 变量计算与综合输出

f-string 不仅可以打印单个变量，还可以展示变量之间计算后的结果。

> **关键逻辑：计算总和**
```Python
my_age = 30
my_height = 100
my_weight = 180
```
> # 计算总和：年龄 + 身高 + 体重
>` total = my_age + my_height + my_weight
> # 使用 f-string 打印计算过程和结果
>` print(f"如果我把{my_age}, {my_height} 和 {my_weight} 加起来我会得到 {total}")

---

### 💡 核心要点总结

1. **代码简洁性**：相比旧式的格式化方法，f-string 让代码更直观，变量名直接出现在它被显示的位置。
2. **动态求值**：f-string 内部的大括号不仅能放变量，还能直接进行数学运算（例如 `{my_age + 10}`）。
3. **注意事项**：确保变量在打印之前已经被赋值，否则程序会报错。
