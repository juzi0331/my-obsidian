
# 📦 Python 数据容器：元组 (Tuple) 详解

> [!ABSTRACT] Python 数据容器：元组 (Tuple) 
> 元组（Tuple）是 Python 中一种重要的线性数据容器。它与列表（List）最大的区别在于其**不可变性**，常用于存储那些不希望被修改的安全性数据。

## 1. 元组的核心特性

- **不可修改**：元组一旦定义完成，其内部元素不可被修改、添加或删除。
- **有序性**：支持通过下标索引访问元素。
- **可重复**：允许存储相同的元素。
- **高效性**：由于其不可变的性质，元组在内存占用和处理速度上通常优于列表。

---

## 2. 元组的定义与创建

### 📝 基本语法

使用圆括号 `()` 定义，元素之间用逗号 `,` 分隔。

```Python
# 标准定义
t1 = (1, 2, 3)

# 定义空元组
t2 = ()
t3 = tuple()

# ⚠️ 重要注意：定义单元素元组
# 必须在元素后加逗号，否则 Python 会将其识别为该元素的原始类型（如整数）
t4 = (100,)  # 这是一个元组
t5 = (100)   # 这是一个 int 类型
```

### 🔍 常用查询方法

由于不可变，元组仅支持查询类操作：

|**方法**|**功能描述**|
|---|---|
|`count(元素)`|统计该元素在元组中出现的总次数。|
|`index(元素)`|查找该元素第一次出现的索引位置。|

## 3. 核心技巧：组包与解包

### 📦 组包 (Packing)

将多个值合并到一个容器（元组或列表）中。

```Python
t = 5, 7, 9, 1  # 自动组包为元组 (5, 7, 9, 1)
```

### 🔓 解包 (Unpacking)

将容器中的元素拆分并赋值给多个变量。

- **基础解包**：变量数量必须与元素数量一致。
```Python
a, b, c, d = (5, 7, 9, 1) # a=5, b=7, c=9, d=1
```
- **扩展解包 (`*`)**：使用星号收集剩余元素，生成一个**列表**。
```Python
x, *y, z = (5, 7, 9, 1) # x=5, y=[7, 9], z=1
*first, last = (1, 2, 3) # first=[1, 2], last=3
```


### 🔄 实战：变量值交换

利用元组的解包特性，可以不使用临时变量直接交换变量值。

```Python
a, b = 10, 20
a, b = b, a  # 本质是将右边组包成元组 (20, 10) 再解包赋值
```

## 4. 综合案例：学生成绩数据分析

在这个案例中，我们使用嵌套元组存储数据，并利用循环和推导式进行处理。

### 📋 数据结构

```Python
students = (
    ("S001", "王林", 85, 92, 78), # 索引: 0-学号, 1-姓名, 2-语文, 3-数学, 4-英语
    ("S002", "李慕婉", 92, 88, 95),
    # ... 更多数据
)
```

### 🚀 统计逻辑分析

1. 计算个人总分与平均分：
    通过 for 循环遍历元组，利用索引取出各科成绩进行累加。
```Python
for i in students:
    total = i[2] + i[3] + i[4]
    avg = total / 3
    print(f"{i[1]} 总分: {total}, 平均分: {avg:.1f}")
```
    
2. 各科横向统计（最高/最低/平均）：
    使用列表推导式提取某一列数据，再配合内置函数 min(), max(), sum()。
```Python
chinese_scores = [i[2] for i in students] # 提取所有人的语文成绩
avg_chinese = sum(chinese_scores) / len(chinese_scores)
```
3. 条件筛选：
    在遍历过程中加入 if 判断（如 if avg > 90）来找出优秀学生。
---

## 💡 总结与建议

- **安全第一**：当你需要向函数传递一组不应被修改的配置信息或固定常量时，请务必使用元组。
- **解包妙用**：在处理函数返回多个值时，解包可以让代码变得非常优雅。
- **性能**：在处理大规模只读数据时，元组比列表更节省内存。
