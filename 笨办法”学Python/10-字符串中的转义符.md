
# 🐍 Python 基础：字符串中的转义符

### 1. 什么是转义字符？

> [!ABSTRACT] Python 基础：字符串中的转义符
在字符串中，某些字符具有特殊含义（如引号用于界定字符串边界）。**转义字符**通过在字符前加上反斜杠 `\`，使该字符被视为内容的一部分，而不是代码逻辑。

---

### 2. 常用转义字符手册

根据您的练习代码，以下是最常用的几种转义方式：

|**转义符**|**含义**|**视觉效果**|**实战案例**|
|---|---|---|---|
|**`\n`**|**换行**|将字符串拆分为两行|`"我被拆分\n成两行了"`|
|**`\t`**|**制表符**|水平缩进（通常为4个空格）|`"\t我被缩进了"`|
|**`\\`**|**反斜杠**|在字符串中显示一个 `\`|`"我是 \\ 一直 \\ 猫"`|
|**`\'` / `\"`**|**引号转义**|在字符串内部显示引号|`"I \"understand\" Joe"`|

---

### 3. 多行字符串 (`"""`)

当需要输入大段带有格式的文字（如清单或长文）时，使用**三引号**是非常方便的选择。

- **特性**：保留代码中的换行和缩进。
- **示例**：
```Python
fat_cat = """
我要列一个清单
\t* 猫粮
\t* 小鱼
\t* 猫薄荷\n\t* 草
"""
```

---

### 4. 代码实战演练

以下是整理后的练习代码，展示了如何通过转义符控制输出排版：

```Python
# 1. 缩进与换行
tabby_cat = "\t我被缩进了"
presian_cat = "我被拆分\n成两行了"
backslash_cat = "我是 \\ 一直 \\ 猫"

# 2. 复合应用 (三引号 + 转义符)
fat_cat = """
我要列一个清单
\t* 猫粮
\t* 小鱼
\t* 猫薄荷\n\t* 草
"""

print(tabby_cat)     # 输出带有缩进
print(presian_cat)   # 输出分为两行
print(backslash_cat) # 显示反斜杠本身
print(fat_cat)       # 输出格式化的清单
```

---

### 💡 核心要点总结

1. **处理引号冲突**：如果字符串边界使用双引号 `"`，内部可以直接写单引号 `'`；但如果要写双引号，就必须使用 `\"` 进行转义。
2. **原始字符串 (r-string)**：如果您不希望反斜杠起作用（例如在处理 Windows 文件路径时），可以在字符串前加上 `r`，如 `r"C:\Users\Name"`。
