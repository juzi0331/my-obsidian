# 🐍 Python 进阶：函数的参数与返回值

> [!ABSTRACT] 🐍 Python 进阶：函数的参数与返回值
在定义函数时，为了让函数能够处理不同的数据并返回计算结果，我们需要引入**参数**和**返回值**机制。
### 1. 函数参数：给函数的“输入”

参数允许我们在调用函数时将外部数据传递到函数内部进行处理。

- **单参数**：如圆的面积计算，仅需半径 $r$。
- **多参数**：如果业务需要多个数据（如长方形的长 $l$ 和宽 $w$），参数之间必须使用**逗号 (`,`)** 分隔。


### 2. 函数返回值：函数的“输出”

使用 `return` 关键字可以将函数内部的计算结果返回给调用者。
- **返回功能**：`return` 仅负责将结果“递出”函数，它本身**不具备打印功能**。
- **结果接收**：调用函数时，通常需要定义一个变量来接收这个返回值，然后配合 `print()` 才能在控制台查看到结果。


### 3. 实战案例：几何面积计算

通过以下两个案例，可以清晰地观察到参数传递与返回值接收的过程：

```Python
# 计算圆的面积  
def circle_area(r): 
# 计算逻辑 
area = 3.14 * r * r 
# 将计算结果返回给外部调用者 return area 
# 调用并接收结果 
c_area = circle_area(10) print(c_area) # 输出: 314.0
```

```Python
def rectangle_area(l, w):
    # 使用逗号分隔多个参数
    area = l * w
    return area

# 调用时按顺序传入数据
r_area = rectangle_area(20, 10)
print(r_area)  # 输出: 200
```

|**知识点**|**描述**|
|---|---|
|**参数传递**|调用时传入的参数数量和顺序必须与定义时保持一致。|
|**`return` 特性**|`return` 语句会立即结束函数的执行并返回结果。其后的代码将不会被执行。|
|**数据接收**|函数返回值可以赋值给变量，也可以直接作为另一个函数的参数使用。|
