# 🐍 Python 基础：多行字符串与换行符

> [!ABSTRACT] Python 基础：多行字符串与换行符
在 Python 中处理多行文本主要有两种方式：使用转义字符 `\n` 或使用三引号 `"""`。

### 1. 换行符 `\n`

`\n` 是一个转义字符，代表 **Newline（换行）**。它允许你在单行定义的字符串内部实现换行效果。

- **代码示例**：
```Python
# 使用 \n 将月份拆分为多行显示
months = "一月\n二月\n三月\n四月\n五月\n六月\n七月"
```
- **适用场景**：适合在简短的字符串中插入少量的换行。

---

### 2. 三引号多行字符串 `"""`

使用三个连续的双引号（或单引号）可以定义多行字符串，这在处理大段文字时非常直观。

- **语法**：`""" 内容 """`。
- **特点**：
    - 可以尽情输入内容，支持 4 行或更多行。
    - 内容中的换行和格式会被原样保留。
- **代码示例**：
```Python
print("""
	这里有一些新鲜的事情正在发生
	通过使用三个双引号
	可以尽情的输入内容
""")
```

---

### 3. 核心对比与注意事项

|**特性**|**换行符 \n**|**三引号 """**|
|---|---|---|
|**定义方式**|在字符串内插入字符|使用三对引号包裹|
|**可读性**|较差（长字符串会变得杂乱）|极佳（所见即所得）|
|**书写要求**|无特殊要求|**三个引号必须写在一起**，中间不能有空格|

---

### 4. 实战代码演练

以下是整合后的代码示例，展示了两种方式的输出区别：

```Python
# 1. 单行定义（不换行）
days = "周一 周二 周三 周四 周五 周六 周日"

# 2. 单行定义（使用换行符）
months = "一月\n二月\n三月\n四月\n五月\n六月\n七月"

print("这里是星期几:", days)
print("这里是几月份:\n", months)

# 3. 多行块定义
print("""
这里有一些新鲜的事情正在发生
通过使用三个双引号
可以尽情的输入内容
甚至可以输入4行~更多行....
""")
```

---

### 💡 核心要点

1. **语法规范**：三引号 `"""` 或 `'''` 必须成对出现，且起始与结束符号必须一致。
2. **转义混用**：在三引号内部仍然可以使用 `\n` 等转义字符，但通常为了代码整洁，直接换行即可。
