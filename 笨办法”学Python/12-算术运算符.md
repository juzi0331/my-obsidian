
# 🐍 Python 基础：算术运算符

> [!ABSTRACT] Python 基础：算术运算符
算术运算符用于执行基础的数学计算，如加、减、乘、除等 2。

### 1. 核心运算符手册

|**运算符**|**描述**|**代码示例**|**结果 (基于 10,4)**|
|---|---|---|---|
|**`+`**|**加**：相加|`10 + 4`|`14` 3|
|**`-`**|**减**：相减|`10 - 4`|`6` 4|
|**`*`**|**乘**：相乘|`10 * 4`|`40` 5|
|**`/`**|**除**：常规除法|`10 / 4`|`2.5` 6|
|**`//`**|**整除**：向下取整|`10 // 4`|`2` 7|
|**`%`**|**取余**：求余数|`10 % 4`|`2` 8|
|**`**`**|**幂**：次方运算|`10 ** 4`|`10000` ($10^4$) 9|

---

### 2. 运算符优先级

当一个表达式中存在多个运算符时，Python 会按照以下优先级顺序执行计算 10：

1. **最高优先级**：`**` (幂运算) 11。
2. **中等优先级**：`*` (乘)、`/` (除)、`//` (整除)、`%` (取余) 12。
3. **最低优先级**：`+` (加)、`-` (减) 13。

> **提示**：如果需要改变运算顺序，请使用**括号 `()`**，括号内的计算优先级永远最高。

#### 🔍 优先级实战案例

计算表达式：`0.1 + 10 / 4 ** 2` 14

1. 首先计算幂：$4^2 = 16$。
2. 接着计算除法：$10 / 16 = 0.625$。
3. 最后计算加法：$0.1 + 0.625 = 0.725$ 15。

---

### 💡 核心点总结

1. **除法类型区分**：
    - `/` 结果始终是**浮点数**（如 `10 / 5` 结果为 `2.0`） 17。
    - `//` 结果会舍弃小数部分，只保留**整数部分** 18。
2. **数据类型转换**：
    - `input()` 接收的内容是字符串，进行数学运算前必须使用 `float()` 或 `int()` 进行类型转换 19。
3. **取余运算的应用**：
    - `%` 运算符常用于判断奇偶数（`n % 2 == 0` 为偶数）或进行周期性循环计算 20。
