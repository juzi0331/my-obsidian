
# 🔄 Python 循环进阶：嵌套循环与实战应用

> [!ABSTRACT] Python 循环进阶：嵌套循环与实战应用
> 嵌套循环是指一个循环体内部又包含了另一个循环。这种结构常用于处理二维数据（如矩阵、表格）或生成特定图形。

---
## 1. 嵌套循环基础结构

在 Python 中，`for` 循环或 `while` 循环都可以进行嵌套。
### 📝 语法示例

```Python
for 外部变量 in 数据集1:
    # 外部循环执行一次
    for 内部变量 in 数据集2:
        # 内部循环会完整执行完所有轮次
        循环体代码
```

> [!TIP] 逻辑关键
> 
> - **外层循环**通常控制“行”或“大轮次”。
>     
> - **内层循环**通常控制“列”或“单行内的细节”。
>     

---

## 2. 经典案例演示

### 📐 案例 1：打印自定义长方形

通过用户输入的长度 $m$ 和宽度 $n$ 来生成图形。

```Python
m = int(input("请输入长度:"))  
n = int(input("请输入宽度:"))  
  
for j in range(n):          # 外层循环控制行数
    for i in range(m):      # 内层循环控制每行打印的 * 数量
        print("*", end=" ") # end=" " 保证打印在同一行
    print()                 # 内层循环结束后换行
```

### ✖️ 案例 2：九九乘法表

利用内层循环的范围随外层循环变化的特性实现三角形排列。

```Python
for i in range(1, 10):          # 外层控制行（1-9）
    for j in range(1, i + 1):   # 内层控制列，随行数增加
        print(f"{j} x {i} = {j * i}", end="\t")
    print("")                   # 换行
```

## 3. 循环控制实战：登录系统

你在代码中展示了两种常见的 `while` 循环逻辑。

### 🔐 场景 A：无限尝试直到成功

使用 `while True` 配合 `break` 语句退出。

```Python
while True:
    username = input("用户名:")
    password = input("密码:")
    if username == "admin" and password == "666888":
        print("登录成功")
        break  # 验证通过，强制终止循环
```

### 🛡️ 场景 B：限制尝试次数

结合计数器变量进行逻辑控制。

- **公式**：剩余次数 $= 5 - \text{count}$
- **逻辑**：当 `while` 条件不满足且未被 `break` 时，执行 `else` 块。

## 💡 总结建议

- **避坑指南**：嵌套循环中，内层循环的变量名（如 `i`, `j`）不要与外层冲突。
- **性能提醒**：尽量减少嵌套层数。每增加一层嵌套，代码执行的复杂度会呈指数级增长。
