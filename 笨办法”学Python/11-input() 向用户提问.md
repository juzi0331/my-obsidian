

# 🐍 Python 基础：使用 `input()` 向用户提问

> [!ABSTRACT] Python 基础：使用 `input()` 向用户提问
在程序中，`input()` 函数是实现**人机交互**的核心，它允许程序暂停并等待用户输入内容。

### 1. 基础用法与特性

- **基本语法**：`变量 = input("提示信息")`。
- **返回值类型**：`input()` 接收到的所有内容都会被处理为 **字符串 (str)** 类型。
- **程序阻塞**：执行到 `input()` 时，程序会停止，直到用户按下回车键。

---

### 2. 提示用户的两种方式

根据您的笔记，有两种常见的方式来引导用户输入：

#### 方式 A：先 `print` 后输入 (灵活控制换行)

通过 `end = " "` 参数，可以让输入框紧跟在提示文字后面，而不是换行。

```Python
print("你多大了", end = " ")  
age = input()  # 用户在提示语后方直接输入
```

#### 方式 B：在 `input` 中直接加入提示语 (简洁)

这是最常用的推荐写法。

```Python
m = input("请输入密码:")  # 提示语直接写在括号内
```

---

### 3. 重要：输入数据的类型转换

由于 `input()` 总是返回字符串，如果需要进行数学运算，必须进行**显式类型转换**。

- **场景**：计算剩余金额。
- **代码逻辑**：

```Python
amt = 10000
je = input("请输入金额: ")
# 必须使用 int() 或 float() 转换，否则无法进行减法运算
print(f"剩余金额: {amt - int(je)}") 
```

---

### 4. 综合实战案例

结合您的练习，这是一个完整的交互流程示例：

```Python
# 1. 收集信息
age = input("你多大了: ")
height = input("你的身高(cm): ")
weight = input("你的体重(kg): ")

# 2. 格式化输出 (使用 f-string)
print(f"所以你{age}岁, 身高{height}cm, 体重{weight}kg")
```

---

### 💡 核心要点总结

|**要点**|**描述**|
|---|---|
|**数据类型**|记住 `input` 拿到的永远是字符串。|
|**转换风险**|使用 `int()` 转换时，如果用户输入了非数字（如 "abc"），程序会报错。|
|**交互友好**|适当使用 `end = " "` 或在提示语末尾加冒号，可以提升用户体验。|
