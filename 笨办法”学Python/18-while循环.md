
# 🔄 Python 基础：while 循环 (while Loop)

> [!ABSTRACT] Python 基础：while 循环 (while Loop)
> 循环的本质是重复执行某件事情。while 循环通过判断一个“条件表达式”来决定是否继续执行代码块：只要条件为 True，循环就会一直持续。

---

## 1. 基础语法结构

理解 `while` 循环的构造是写出正确代码的第一步。

```Python
while 条件表达式:  # True 则执行，False 则跳出
    循环体语句 1
    循环体语句 2
else:
    条件为 False，循环正常结束时执行
```

### 📍 关键点解析

- **条件表达式**：通常是一个比较运算（如 `i < 10`），结果为布尔值。
- **缩进**：属于循环体的代码必须保持统一的缩进。
- **else 子句**：这是一个可选部分，只有在循环**正常结束**（即条件变为假）时才会运行。
---

## 2. 计数循环的三要素

当你需要执行固定次数的循环时，必须具备以下三个要素，缺一不可：

1. **初始变量**：设置计数器的起始值（如 `i = 0`）。
2. **循环条件**：设置终止界限（如 `i < 10`）。
3. **变量更新**：在循环体内修改计数器（如 `i += 1`），否则会导致**死循环**。
---

## 3. 实战案例演练

### 📝 案例 1：重复打印指定次数

> **需求**：循环 10 遍打印 "人生苦短, 我用 Python"。


```Python
i = 0  # 1. 初始值
while i < 10:  # 2. 循环条件
    print(f"人生苦短,我用Python,第{i}遍")
    i += 1  # 3. 变量更新
else:
    print("循环结束")
```

### 🔢 案例 2：数学累加计算

> **需求**：计算 1-100 之间的偶数累加之和。

- **逻辑思路**：
    1. 准备一个 `sum` 变量存放结果，初始为 $0$。
    2. 准备一个计数器 `i`，从 $1$ 开始。  
    3. 在循环中判断 `i % 2 == 0` 是否成立（即是否为偶数）。
---

## 💡 新手避坑指南

> [!CAUTION] 警惕“死循环”
> 
> 如果你忘记写 i += 1，i 的值将永远保持不变，while i < 10 将永远为 True。这会导致程序卡死并疯狂占用电脑内存。

> [!TIP] 关联建议
> 
> 学完 while 循环后，建议对比学习 [[19-for循环]]。while 更适合不确定循环次数的场景，而 for 更适合遍历固定数据集。
