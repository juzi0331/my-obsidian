
# 🐍 Python 基础：字符串组合与输出技巧

> [!ABSTRACT] Python 基础：字符串组合与输出技巧
在 Python 中，我们可以通过多种灵活的方式来处理和组合字符串，以满足不同的排版需求。

### 1. 使用 `.format()` 进行动态填充

这是格式化字符串的一种常用方法。通过在字符串中使用 `{}` 作为**占位符**，我们可以将指定的内容填充进去。

- **代码示例**：
```Python
# "{}" 是占位符，format("雪") 将内容填充到该位置
print("它的羊毛白的像{}".format("雪")) 
```

- **优点**：可以将字符串模板与实际数据分离，使代码结构更清晰。

---

### 2. 字符串乘法 (重复输出)

在 Python 中，字符串可以与整数相乘，表示将该字符串重复输出指定的次数。

- **场景**：常用于打印分隔符或简单的图形装饰。
- **示例**：`print("." * 10)` 会输出 10 个连续的点号。

---

### 3. 字符串拼接与换行控制

通过 `+` 运算符和 `print()` 函数的 `end` 参数，我们可以精确控制输出的内容和格式。

#### ① 拼接字符串 (`+`)

使用 `+` 运算符可以将多个字符串变量连接成一个更长的字符串。

- **示例**：`end1 + end2` 会将“你”和“好”拼接在一起。

#### ② 控制结束符 (`end`)

默认情况下，`print()` 函数在执行完后会自动换行。通过设置 `end` 参数，我们可以改变这一行为。

- **代码示例**：
```Python
# 使用空字符串作为结束符，避免换行，使下一次打印内容接在后面
print(end1 + end2, end = "") 
```

---

### 4. 实战代码演练

以下是整合后的完整示例，展示了这些技巧如何协同工作：

```Python
# 1. format 方法应用
print("玛丽有一直小羊羔")
print("它的羊毛白的像{}".format("雪"))

# 2. 字符串乘法 (打印分隔符)
print("." * 10)

# 3. 变量拼接与换行控制
end1 = "你"
end2 = "好"
end3 = "Python"

# 拼接并保持在同一行输出
print(end1 + end2, end = " ")
print(end3)
# 最终输出结果：你好 Python
```

---

### 💡 核心要点

1. **数据类型限制**：字符串拼接 (`+`) 只能用于字符串之间。如果要拼接数字，必须先用 `str()` 进行转换。
2. **灵活的 `end`**：`end` 参数不仅可以设为空字符串 `""` 避免换行，还可以设为逗号 `,`、空格 `" "` 或其他字符，作为输出项之间的连接符。
