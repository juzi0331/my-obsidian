# 🐍 Python 基础：变量的使用

> [!ABSTRACT] Python 基础：变量的使用
变量是程序中用于存储数据的“容器”。通过给数据起一个名字（变量名），我们可以在程序中方便地引用和修改它。

### 1. 核心概念：赋值运算符

在 Python 中，单等号 **`=`** 被称为**赋值运算符**。

- **作用**：将等号右边的值赋给左边的变量名。
- **示例**：`cars = 100` 表示创建了一个名为 `cars` 的变量，并存储了数值 `100`。

---

### 2. 实战案例：拼车逻辑计算

通过定义基础变量，我们可以利用数学运算推导出新的信息。

#### 🔹 基础数据定义

```Python
cars = 100                # 汽车总数
space_in_a_car = 4        # 每辆车的座位数（不含司机）
drivers = 30              # 司机数量
passengers = 90           # 乘客总数
```

#### 🔹 逻辑计算公式

基于上述变量，我们可以进行如下运算：

- **闲置车辆**：
$$cars\_not\_driven = cars - drivers$$
- **总运输能力**：
 $$carpool\_capacity = drivers \times space\_in\_a\_car$$
    
- **平均乘客分配**：
$$average\_passengers\_per\_car = passengers / drivers$$
---

### 3. 常见问题与代码规范

为了编写高质量的代码，需要注意以下细节：

|**关注点**|**说明**|**正确示例**|
|---|---|---|
|**`=` vs `==`**|**`=`** 用于赋值；**`==`** 用于检查左右两边是否相等。|`x = 100` (赋值)|
|**空格规范**|在操作符（如 `=`、`+`）两边加上空格会使代码更容易阅读。|`x = 100` (优于 `x=100`)|
|**浮点数转换**|如果将整数写成小数形式（如 `4.0`），变量类型会变为**浮点数**。|`space = 4.0`|

---

### 💡 核心要点总结

1. **所见即所得**：在 `print` 函数中，直接放入变量名会输出其存储的值。
    - 示例：`print("这里有", cars, "辆小汽车")`。
2. **动态更新**：变量的值可以随程序运行而改变，你可以利用旧变量的值来计算并更新新变量。
3. **命名建议**：使用具有描述性的变量名（如 `cars_driven` 而不是简单的 `c`），能显著提升代码的可读性。
