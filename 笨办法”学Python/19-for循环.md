
# 🔄 Python 基础：for 循环 (for Loop)

> [!ABSTRACT] Python 基础：for 循环 (for Loop)
> for 循环是一种遍历循环，主要用于从头到尾访问“待处理的数据集”（如字符串、列表、range 序列等）中的每一个元素，并对元素进行处理。

---

## 1. 基础语法结构

`for` 循环的逻辑非常直观：它会自动取出数据集中的每一个元素，赋值给临时变量，然后执行循环体。

```Python
for 临时变量 in 待处理的数据集:
    循环体代码(对变量进行处理)
else: # 可选
    循环正常结束时运行的代码
```

### 📍 重点解析

- **临时变量**：每循环一次，变量都会被更新为数据集中的下一个值。
- **数据集**：必须是可迭代的（如字符串、range 序列等）。
- **else 块**：只有当循环**正常遍历完**所有元素后才会执行；如果循环中途被 `break` 跳出，则不执行。
---

## 2. 核心工具：range() 函数

在 `for` 循环中，我们经常使用 `range()` 来产生一个整数序列。

- **语法**：`range(start, stop)`
- **注意**：生成的序列包含 `start`，但**不包含** `stop`（左闭右开）。
    - 例如：`range(1, 101)` 生成的是从 $1$ 到 $100$ 的整数。  

---

## 3. 实战案例演练

### 🔤 案例 1：遍历字符串

逐个打印字符串中的每一个字符。

```Python
msg = "Hello-python"
for i in msg:
    print(i)
else:
    print("循环结束")
```

### 🔢 案例 2：计算 $1-100$ 之间所有奇数之和

利用取模运算 `%` 判断奇偶性。

```Python
total = 0
for i in range(1, 101):  # 生成 1 到 100 的序列
    if i % 2 == 1:       # 判断是否为奇数
        total += i
print(f"1-100奇数之和为: {total}")
```

### ➗ 案例 3：计算 $100-500$ 之间所有 3 的倍数之和

逻辑与奇数求和类似，只需修改判断条件。

```Python
total = 0
for i in range(100, 501): # 注意：结束值要写 501 才能包含 500
    if i % 3 == 0:        # 判断是否能被 3 整除
        total += i
print(f"100-500之间3的倍数之和为: {total}")
```

---
## 💡 技巧与常见错误

> [!TIP] 逻辑小贴士
> 
> 1. **求和思想**：先定义一个初始为 $0$ 的变量（如 `total`），然后在循环中不断累加。
>     
> 2. **边界问题**：`range(1, 100)` 只到 $99$。如果你需要计算到 $100$，记得写成 `range(1, 101)`。
>     
> 3. **缩进**：循环体内的代码必须有统一的缩进，否则 Python 会报错。
>
