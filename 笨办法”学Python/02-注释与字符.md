
# 🐍 Python 基础：注释与 `#` 字符

> [!ABSTRACT] Python 基础：注释与 `#` 字符
在编程中，注释是写给“人”看的，而不是给“机器”看的。它是代码中极其重要的一部分，能显著提升程序的可读性。

### 1. 什么是注释？
- **定义**：在 Python 中，`#` 号后面的所有内容都会被解释器忽略。
- **特性**：注释不会被执行，也不会影响程序的运行结果。
---

### 2. 注释的三大核心用途

根据您的代码练习，注释主要有以下三种用法：

|**用法**|**描述**|**示例**|
|---|---|---|
|**解释说明**|帮助开发者理解代码的功能和逻辑。|`# 这里打印欢迎语`|
|**行内注释**|写在代码行末尾，对当前行进行解释。|`print("Hello") # 打印内容`|
|**代码屏蔽**|在调试时临时让某行代码失效，而不必删除它。|`# print("暂时不跑这行")`|

---

### 3. 注释的实战示例

以下是整合后的练习代码，展示了注释在不同位置的表现：

```Python
# 1. 独立行注释：注释可以帮助你更好的阅读程序
# 在 # 后面的所有内容将被 Python 忽略
print("我可以这么写代码.") # 2. 行内注释：这里的注释不会被解析

# 3. 屏蔽代码：通过注释，我们可以阻止某些代码运行
# print("这段代码不会被执行.")

print("这段代码可以被执行")
```

---

### 💡 核心总结

1. **提升理解**：写好注释能帮助自己和团队成员更快地看懂代码。
2. **调试利器**：利用注释进行代码排错是开发中最常用的技巧之一。
3. **团队和谐**：规范且清晰的注释是团队协作的重要基础。
