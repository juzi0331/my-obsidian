
# 🐍 Python 基础：逻辑运算符

> [!ABSTRACT] Python 基础：逻辑运算符
逻辑运算符用于将多个布尔表达式组合成一个最终的布尔值（`True` 或 `False`）。3

### 1. 三大核心运算符

Python 主要包含三种逻辑运算符：4

|**运算符**|**逻辑名称**|**描述**|**示例**|
|---|---|---|---|
|**`and`**|**与**|两个条件**同时**为真，结果才为 `True` 5|`A and B`|
|**`or`**|**或**|两个条件中**只要有一个**为真，结果就为 `True` 6|`A or B`|
|**`not`**|**非**|**取反**：真变假，假变真 7|`not A`|

---

### 2. 真值表 (Truth Table)

这是理解逻辑运算最直观的方式：

|**表达式**|**结果**|**备注**|
|---|---|---|
|`True and True`|**True**|全真才真|
|`True and False`|**False**|一假即假|
|`True or False`|**True**|一真即真|
|`False or False`|**False**|全假才假|
|`not True`|**False**|反转|

---

### 3. 实战案例：区间判断

根据你提供的代码 8，逻辑运算符在处理数学区间时非常高效。

#### 🔹 场景一：判断数值是否在闭区间内

若要判断变量 $n$ 是否满足 $10 \le n \le 20$：9

- **代码实现**：`n >= 10 and n <= 20` 10
- **逻辑解析**：必须同时满足“不小于10”和“不大于20”两个条件。 11

#### 🔹 场景二：判断数值是否在区间之外

若要判断变量 $k$ 是否处于 $10$ 到 $20$ 的范围之外：12

- **代码实现**：`k < 10 or k > 20` 13
- **逻辑解析**：只要满足“小于10”或者“大于20”其中之一即可。 14

---

### 💡 进阶优化建议

1. 链式比较 (Python 特色)：
    在 Python 中，判断区间 $10 \le n \le 20$ 有一种更符合数学直觉的写法：
```Python
 print(10 <= n <= 20)  # 这与 n >= 10 and n <= 20 等价
```
    
2. **短路求值 (Short-circuit Evaluation)**：
    - 在 `A and B` 中，如果 `A` 为 `False`，Python 不会再计算 `B`。  
    - 在 A or B 中，如果 A 为 True，Python 不会再计算 B。  
        这在编写高效代码或防止报错（如除以零检查）时非常有用。
