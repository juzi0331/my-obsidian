## 🐍 Python 基础：手动格式化字符串 (`.format`)

> [!ABSTRACT] Python 基础：手动格式化字符串 (`.format`)
在 Python 中，`.format()` 方法是一种灵活的字符串格式化方式。它通过在字符串中使用大括号 `{}` 作为**占位符**，随后在调用时按顺序填充具体内容 1。

### 1. 基础语法与复用

您可以先定义一个包含占位符的“模板”变量，然后在不同的场景下多次调用它 2。
- **定义模板**：`formatter = '{} {} {} {}'` 3。
- **填充内容**：调用 `formatter.format(...)` 并传入对应数量的参数 4。

---

### 2. 多样化的数据支持

`.format()` 方法非常强大，它可以接受任何数据类型作为参数并自动将其转换为字符串形式 5：

|**填充类型**|**示例代码**|**输出结果 (示意)**|
|---|---|---|
|**整数**|`formatter.format(1, 2, 3, 4)`|`1 2 3 4`|
|**字符串**|`formatter.format("one", "two", "three", "four")`|`one two three four`|
|**布尔值**|`formatter.format(True, False, False, True)`|`True False False True`|
|**自身嵌套**|`formatter.format(formatter, formatter, formatter, formatter)`|`{} {} {} {} ...` (重复四遍)|

---

### 3. 代码进阶：多行书写与排版

当填充的内容较长时（例如诗歌或长句），建议将 `.format()` 的参数分行书写，以增强代码的可读性 7。

```Python
# 这种写法清晰地展示了每个占位符对应的内容 
print(formatter.format(
    "将这里的字符串",
    "替换为自己的内容",
    "可以是一首歌",
    "可以是一首诗"
))
```

---

### 💡 核心要点总结

- **顺序匹配**：大括号 `{}` 的数量必须与 `.format()` 中传入的参数数量一致，否则会抛出错误 8。
- **灵活性**：与 `f-string` (变量名直接放入大括号) 不同，`.format()` 更适合在**运行前尚不确定具体变量名**，或需要重复使用同一个**格式化模板**的场景 9。
- **嵌套能力**：你可以把一个包含占位符的字符串传递给另一个占位符，实现动态的模板构建 10。
